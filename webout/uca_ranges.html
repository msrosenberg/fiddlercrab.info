The first map shows the approximate density of species richness, with denser color where more species are found. The second map shows approximate point locations where fiddler crabs have been recorded in the scientific record. Red markers indicate points where fiddler crabs are found; purple indicates fossil-only locations; blue indicate false location records. See the location index for a full list of all point locations.
